Ridgeline Junk Removal is a locally owned service dedicated to keeping the Pacific Northwest clean. This repository contains the source code and infrastructure configuration for our high-performance web platform.

## 🌐 Modern Cloud Infrastructure
To ensure our customers in the Portland metro area have the fastest possible experience, I recently migrated the entire site to a modern cloud stack:

* **Hosting:** Migrated from Netlify to **Cloudflare Pages** to leverage a global edge network and eliminate bandwidth caps.
* **DNS Management:** Securely managed via Cloudflare following a full nameserver migration from Namecheap.
* **Business Email:** Integrated with **iCloud Mail** for custom domain support. Configured and verified specialized DNS records (**MX, SPF, and DKIM**) to ensure 100% email deliverability.
* **Performance:** Enabled **Brotli compression** and **Auto-Minification** of assets to reduce page load times to under 1 second.

## 🌲 Local Service Focus
* **Area:** Serving Portland, Beaverton, Happy Valley, and the surrounding areas.
* **Mobile-First:** Designed for easy booking on mobile devices for customers on-the-go.
* **SEO:** Built with local schema markup to improve visibility in Portland search results.

## 🛠 Tech Stack
* **Frontend:** HTML5, CSS3 (Responsive Design)
* **CI/CD:** GitHub Actions for automated deployment
